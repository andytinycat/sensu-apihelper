# Sensu::Apihelper

This gem is a simple wrapper around the [Sensu](https://github.com/sensu/sensu) API.

## Installation

Add this line to your application's Gemfile:

    gem 'sensu-apihelper'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install sensu-apihelper
